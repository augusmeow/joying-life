# [Joying Life](https://augusmeow.github.io/joying-life/)

## Develop

prerequisites：

- nodejs(Version >= 16)
- yarn(Version >= 1) (`npm install -g yarn`)

git clone：

```git
git clone https://github.com/augusmeow/joying-life.git
```

run：

```yarn
yarn install
yarn src:dev
```

## Contribute

Before you commit, please run `vuepress-vite build src`
